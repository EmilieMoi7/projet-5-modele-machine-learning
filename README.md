# Projet 5 – Modèle de machine learning

## Description
Ce projet est réalisé dans le cadre du Projet 5 de la formation AI Engineer.
L’objectif est de mettre en place et déployer un modèle de machine learning.

Cette première étape du projet concerne la mise en place de l’environnement
de travail et du système de gestion de versions avec Git et GitHub.

## Objectifs
- Mettre en place un dépôt Git structuré
- Assurer le suivi et la traçabilité des modifications
- Préparer une base propre pour la suite du projet

## Structure du projet
- src/ : code source du projet
- tests/ : tests unitaires
- notebooks/ : notebooks d’exploration
- data/ : données du projet
- requirements.txt : dépendances Python
